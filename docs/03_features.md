# Features

## Reader
- Add / Edit / Delete
- Search by multiple fields
- Classify (Student, Lecturer, Other)

## Library Card
- Issue card
- Barcode
- Expiry & Renewal

## Book
- CRUD book
- Manage quantity
- Track damaged/lost

## Borrow/Return
- Borrow book
- Return book
- Track borrow history

## Report
- Borrow statistics
- Inventory report
- Lost/damaged report

## System
- Role-based access (Admin, Staff)
