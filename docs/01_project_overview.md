# Library Management System

## Overview
Hệ thống quản lý thư viện giúp quản lý sách, bạn đọc và quá trình mượn/trả.

## Tech Stack
- ASP.NET Core MVC
- Entity Framework Core
- SQLite

## Architecture
- MVC Pattern
- Service Layer
- Repository Pattern (optional)

## Modules
- Book Management
- Reader Management
- Library Card
- Borrow/Return
- Report & Statistics
